# Boardwalk Games

A responsive multi-page website for a fictional board game cafe, designed to showcase services, events, and a game library while allowing users to submit booking inquiries.

## Overview
Boardwalk Games is a frontend portfolio project built to practice creating a cohesive multi-page user experience with semantic HTML, custom CSS, and Bootstrap components. The site helps visitors explore the venue, browse featured games, and submit a booking inquiry through a clear call-to-action flow.

## Features
- Responsive, multi-page layout with consistent navigation across all pages.
- Homepage sections for services, events, and contact details.
- Dedicated game library page with categorized game cards and carousel visuals.
- Booking inquiry form with required fields and a success/confirmation page.
- Accessible structure with semantic elements, descriptive alt text, and mobile-friendly viewport setup.

## Technologies
### Technologies Used
- **Languages:** HTML5, CSS3

## How to Use

### Local Setup
```bash
git clone https://github.com/<your-username>/boardwalk-games.git
cd boardwalk-games
```

Open `index.html` directly in your browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
